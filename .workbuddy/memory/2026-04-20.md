## 网站已完成（v0.1）

**技术栈**：Vite + React + TS + Tailwind v3 + Zustand（localStorage 持久化）

**项目结构**：仓库根目录就是 Vite 项目
- `src/types.ts` — 数据模型
- `src/seed.ts` — 默认数据（4 款豆子 + 新包装件清单 + 成本项预置）
- `src/engine.ts` — 计算引擎（packCost / suggestedPrice / computeProfit / Break-even）
- `src/store.ts` — Zustand store + 导入导出
- `src/verify.ts` — 20 条单元测试（`npm run verify`），**全部通过**
- `src/pages/` — 6 个 Tab：CostItems / Ratios / Capacity / Beans / Profit / Audit
- `src/components/PasswordGate.tsx` — 前端密码门（sha256，默认密码 `dhj123`）
- `.github/workflows/deploy.yml` — Push main → 自动 build + 部署 GH Pages；密码哈希通过 `VITE_PASSWORD_HASH` secret 注入
- `vite.config.ts` base = '/DHJ_Pricing/'

**核心公式（最终版）**：
```
生豆用量 = 熟豆g / (1-lossSort) / (1-lossRoast)
生产成本 = (生豆 + 包装) × (1+lossPack) + 物流
售价 = 生产成本 / (1 - margin - platformFee - marketingShare)
     ← 保证扣除平台和营销后毛利率正好 = margin
净利润 = GMV - 年生产成本 - 运营分摊 - 平台抽成 - 营销 - 退货计提
Break-even kg/年 = 年度总运营 / 加权每kg贡献边际
```

**测试结果（Mid 产能 1320kg，50% 毛利）**：年 GMV ≈146 万，净利润 ≈69 万，净利润率 47%，保本月销量 3.8 kg。
